Spring Boot - Validation Com o Hibernate Validator realizar validações fica muito simples. Ao adicionar validações no seu modelo, você impede que o usuário envie para o back-end valores não desejados. Alguns exemplos de anotações são @NotNull, @NotEmpty, @Size, entre outros. É importante lembrar de adicionar o @Valid na classe Controller e de importar a dependência Validation! https://www.devmedia.com.br/hibernate-validator-validando-dados-no-back-end-com-anotacoes/37979
